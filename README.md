# Kepler Orbit Lab

AP 일반물리학 후속 탐구용 정적 웹사이트입니다.

## 실행
`index.html`을 브라우저에서 열거나 GitHub Pages에 `index.html`, `style.css`, `app.js`를 업로드합니다.

## 기능
- Newton–Raphson + bisection fallback
- 실제 천체 근사 궤도 애니메이션
- 반복 과정과 잔차 표시
- Kepler 제2법칙, 에너지, 각운동량, 제3법칙 검증
- 수렴 heatmap과 초기값 전략 비교
- CSV 저장

## 범위
이상적 2체 타원 궤도만 다룹니다. 섭동, 상대론적 세차, 3차원 궤도요소, 정밀 천체력은 포함하지 않습니다.
